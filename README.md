REST API using FastAPI and Imagekit.io

## Features
- User registration and login with JWT authentication
- Create, Read, Update, Delete (CRUD) operations for Posts
- Protected routes accessible only to authenticated users
- Async-compatible API endpoints
- Images and Videos Saved on Imagekit.io
- Feed With all user Posts
- 
## Tech Stack
- **Backend:** FastAPI, Python 3.11+
- **Database:** SQLite with SQLAlchemy ORM
- **Authentication:** FastAPI JWT implementation
- **Libraries:** Pydantic, Async SQLAlchemy
